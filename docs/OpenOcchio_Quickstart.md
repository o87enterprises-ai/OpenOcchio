# 🔍 OpenOcchio: Comprehensive Quickstart Guide

OpenOcchio is a consumer-grade AI confidence indicator that adds a "verification layer" to your AI interactions. It helps you distinguish between factual AI responses and "hallucinations" using semantic consistency and logprob analysis.

---

## 🛠 1. Local Setup (macOS)

To run the local tools (Overlay or CLI), you must first activate the Python environment.

### Step A: Activate/Create Virtual Environment
From your project root:

**If the venv exists:**
```bash
source confidence_wrapper/venv/bin/activate
```

**If the venv is missing (Fresh Install):**
```bash
python3 -m venv confidence_wrapper/venv
source confidence_wrapper/venv/bin/activate
pip install -r requirements.txt
```

### Step B: Start the "Judge" (Ollama)
OpenOcchio uses a local LLM to verify external AI responses.
1. Download Ollama from [ollama.com](https://ollama.com).
2. Run the specialized model:
   ```bash
   ollama run qwen2.5:3b
   ```

## 🚀 2. Use Cases & Workflows

### Use Case 1: The "Clipboard Spy" (Local Web AI Testing)
Perfect for checking **Claude, ChatGPT, or Gemini** in your browser.
1. **Launch Overlay**: 
   ```bash
   python3 confidence_pro/system_overlay.py
   ```
2. **The Workflow**:
   - Ask Claude a question in Safari/Chrome.
   - **Copy (Cmd+C)** Claude's response.
   - The floating OpenOcchio gauge will immediately update with a confidence score based on the text in your clipboard.

### Use Case 2: The Mobile PWA (Zero-Terminal Experience)
Access OpenOcchio on your iPhone/Android via a standalone app.
1. **Setup Backend**: Deploy the `openocchio-backend` folder to Hugging Face Spaces (Docker).
2. **Connect Frontend**: Update `confidence_pro/script.js` with your Space URL.
3. **Install**: Open your GitHub Pages URL on your phone and tap **"Add to Home Screen"**.
4. **The Workflow**: Type a prompt directly into the mobile app to see how confident Llama 3 (via Groq) is about the answer.

### Use Case 3: The CLI Wrapper (Developer Tool)
Quickly check a specific prompt or string from the terminal.
1. **Command**:+
   ```bash
   python3 confidence_wrapper/wrapper.py "Is the earth flat?"
   ```
2. **Result**: Displays a visual color-coded confidence bar.

## 📊 3. Interpreting the Scores

- **80% - 100% (Green)**: **Confident.** The AI is consistent and the math checks out.
- **60% - 79% (Yellow)**: **Steady.** Likely correct, but minor semantic variations detected.
- **40% - 59% (Orange)**: **Unsure.** The AI is showing signs of hesitation or inconsistency.
- **0% - 39% (Red)**: **Insecure.** High probability of hallucination or refusal.
